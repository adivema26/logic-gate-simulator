# Logic Gate Simulator using Logisim

## Overview
This project simulates basic logic gates:
- AND
- OR
- XOR

Designed using Logisim Evolution.

## Software Used
- Logisim Evolution
- GitHub

## Features
- Interactive input testing
- Truth table verification
- Separate circuits for each gate
